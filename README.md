# module-algorithm-interview

OpenSkill Galaxy · **数据结构与算法刷题** 模块

面向校招、实习、转码、编程考试、算法入门和技术面试的学习者，提供从数据结构基础到经典算法套路的静态课程、刷题、解析、模拟面试和案例训练模块。

**声明**：所有题目均为原创模拟练习（source_type = `curated-generated`）。

## 访问地址

- 线上：https://openskill-galaxy.github.io/module-algorithm-interview/
- 仓库：https://github.com/openskill-galaxy/module-algorithm-interview
- 总入口站：https://openskill-galaxy.github.io/

## 技术栈

Vite + React + TypeScript + Tailwind CSS + React Router + Zustand + Fuse.js + 静态 JSON + GitHub Actions Pages。无后端、无数据库。
